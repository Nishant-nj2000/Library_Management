Add Book

- This will add a book directly to the database alongwith the details you provide while adding a new book.

Import Book

- This will import the available quantities of specific book present in the api to the database.

-This can also import specific number of random books present in the api to the database, if no parameters were given.

Update Book

- This will open a modal box with details of particular book already filled, the user can edit any of the fields submit the same for getting the book updated.

Delete Book

- This will delete the book permanently from the database.

Add Member

- This will add a member directly to the database alongwith the details you provide while adding a new member.

Update Member

- This will open a modal box with details of member already filled, the user can change any of the details and submit the same for updating the member details.

Delete Member

- This will This will delete the member permanently from the database.

Issuing a Book

- The book name can be selected from the autocomplete dropdown.

- Then the member name can be selected from another autocomplete dropdown.

- Clicking submit will issue the book to the particular member while issuing a book.

Returning a Book

- Once the member returns a book two options for Rent Paid will be asked Yes or No.

- If Yes, then the no outstanding amount will be recorded for the user.

- If No, then the particular amount to be collected will be transfered to the outstanding amount.

Outstanding Amount Settlement

- The outstanding amount settlement can be cleared by timely partial payments and complete payment at once.

Terms

- Same member cannot be issued a particular book for the second time if not returned the same by him/her.

- A member's outstanding amount should not be greater than 500 otherwise the book cannot be issued to that member.

- A member can only be issued 2 book at once. After that return a book to issue another.
